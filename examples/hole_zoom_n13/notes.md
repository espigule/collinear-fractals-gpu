# n=13 finite-capture hole zoom

This example records metadata for a finite-capture zoom near an \(n=13\) hole,
where \(N = 2n-1 = 25\).

- Parameter: \(c = 2.0719 + 3.0537i\).
- Expected verdict: exploratory until a verified certificate is exported.
- Status: exploratory.

To reproduce, open the browser explorer, set `n = 13`, set `k_max = 37`, and
zoom near the listed parameter. Replace `certificate.json` with an exported
finite-search certificate once the curated example is finalized.

The visualization is intended to be visually memorable, but theorem-level
proofs do not rely on visual inspection.
